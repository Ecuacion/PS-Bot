Features


These are the major features of this bot that set it apart from most other PS bots.

  • Chat room logging: this feature of the bot will log the chats the bot is in, if enabled in Config. The users can then request logs of a chat as they wish specifying the server, room and date via the viewlogs command.
  • Being able to connect to multiple servers: this allows the user to preset what servers they want the bot to connect in via Config. This allows the bot to be able to do more cross server commands that most other bots cannot do.
  • Tracking when a user was last seen across several servers: because this bot has the capability of connecting to several servers, it allows it to store when a user was last seen with that in consideration.
  • Offline messaging (tells) across servers: this feature allows the bot to be able to send offline tell messages across servers. It will send the user in question the tell once they join a server/room the bot is in.

Commands


Here's a brief overview of the commands this bot has:
  • Informational Commands:
    • seen [user] - displays when [user] was last seen in the bots database.
    • about - displays the link to the main GitHub for this bot with the author.
    • uptime - displays how long the bot has been started for.
    • memusage - displays how much RAM the bot is using / taking up.
    • regdate [name] - displays when [name] was registered on Pokémon Showdown!.
    • viewlogs [server], [room], [DD-MM-YYYY] - displays the chat logs for the respected room on the specified date.
  • Developer Commands:
    • eval [target] - executes [target].
    • kill - kills the bot process.
    • reload - hotpatches commands for the bot.
    • permission [permission], [rank] - sets [permission] to be for the [rank] rank.
    • reconnect - reconnects the bot for the respective server.
  • Misc. Commands
    • custom [target] - makes the bot say [target].
    • join [room] - makes the bot try to join [room].
    • leave - makes the bot leave the room this command is done in.
    • say [target] - makes the bot say [target].
    • pick [choice], [choice], ... - picks a random [choice]. Requires at least two options.
    • helix - gives a random magic 8 ball type reply.
